# 基本原則（最終整理）

* 言語は「発しながら作られるもの」である。
* 即時文法は、発話の順番や感情の入り込みなど、**時間の流れに沿った言語使用**を記述する。
* 調整文法は、統語的構造に基づき、**時間をかけて組み立てられる言語使用**を記述する。
* プロセス文法モデルは、**System 1（即時）／System 2（調整）／Time（順序とプロセス）** の三要素で構成される。
* 即時性と調整性は連続体であり、発話は両者のバランスによって成立する。
* 即時文法は、聞き手の反応を含む相互作用を前提とし、あいづち・呼びかけ・修正・繰り返しなどを文法的要素として扱う。


### 基本原則の中の基本原則

* **時間の概念が必須である。**

  * 即時文法は「順番としての時間」に依拠する。
  * 調整文法は「構造を組み立てるための時間」に依拠する。
  * どちらの文法も、時間を切り離しては存在しない。


### 時間は、「即時文法＝順番」「調整文法＝構造」の両者を貫く必須の概念である。

## 即時文法と調整文法の対照表（例）

| 特徴     | 即時文法（System 1）        | 調整文法（System 2）     |
| ------ | --------------------- | ------------------ |
| ルール    | 発話の順番                 | 統語的構造（シンタクス）       |
| 時間性    | その場で瞬時に生成             | 熟慮的に構築             |
| 典型例    | 「はい」「そう」「おみせ、いった、きのう」 | 「はい、私はきのうお店に行きました」 |
| 文末表現   | 「よ」「ね」「さ」など終端マーカー     | 述語終止による文完成         |
| 文の完成度  | 不完全でも成立（補い合いで理解）      | 完結性を重視             |
| 主な機能   | 相槌、呼びかけ、感情表出、訂正、繰り返し  | 論理的説明、記述、文章化       |
| 学習上の注意 | 「非文」ではなく自然な言語使用       | 文法教育で重視されてきた部分     |
